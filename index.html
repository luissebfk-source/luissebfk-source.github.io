Computer Hardware Explained:
Core Components & Cards

A quick reference to the major parts inside a computer.

CPU (Processor)

Role
Computes binaries to run the OS, apps, game logic.
Key Specs
Cores/Threads|Base/Boost clocks|Cache|TDP.
Examples
AMD Ryzen 5 5600|Intel Core i5-12400F
iGPU
Integrated graphics on some CPUs (Intel UHD|AMD Vega)
Overclocking
Unlocked models (AMD X|Intel K) can be overclocked via BIOS or software, requires good cooling
Thermals
Keep under sustained limits for best performance
Socket
Must match motherboard (AMX, LGAXXXX, etc.)

Practical Notes

Socket
AMX/LGAXXXX (CPU must match MB socket)
Use-Cases
Productivity|gaming|compiling|streaming.
Bottlenecks
Pair appropriately with GPU to avoid CPU-bound frames.
Cooling
Stock coolers for non-overclocked; aftermarket for better thermals/noise
Thermal Paste
Improves heat transfer between CPU and cooler, liquid metal is better but conductive
Installation
Align triangle markers, secure lever, apply paste, mount cooler firmly
Thermal paste application
Pea-sized dot in center or thin line, spreads under pressure; avoid excess,
sometimes an X shape can be adivced

GPU (Graphics Card)

Role
Parallel processing for graphics, AI, compute.
Key Specs
CUDA cores (NVIDIA)|Stream processors (AMD)|Tensor/RT cores|VRAM|boost clock|TBP.
Examples
NVIDIA GTX/RTX/Quadro Series | AMD RX/Pro | iGPU (APU)
VRAM
4GB minimum for 1080p, 8GB+ for 1440p, 10-12GB+ for 4K
PCIe
Typically x16 slot, Gen3/4/5 (backward compatible)
Legacy
AGP (old), PCIe x1 (low-end)
Cooling
Air (blower/tower) or Liquid (AIO/custom)
Power
6-pin/8-pin/12VHPWR connectors; check PSU requirements.
Overvolting & Undervolting
Can improve performance and lifespan,
overvolting can allow higher clockspeeds, on spesific models but may void warranty

Display Outputs

Outputs
Composite|VGA|DVI|HDMI 2.1|DisplayPort 1.4/2.1
Notes
Match refresh rate & resolution;
check PSU headroom and cable version.
Multi-GPU
SLI/CrossFire mostly deprecated, use single powerful GPU.
eGPU
External GPU via Thunderbolt 3/4 (laptop use)
Ray Tracing
Real-time lighting effects, requires RTX/Pro series or AMD RX 6000 series and up
DLSS/FSR
AI upscaling techniques for better performance
Overclock
Boost clocks via software or GPU BIOS mods, must monitor temps and stability

Memory (RAM)

Type
DDR3|DDR4|DDR5
Capacity
Typical 16-32GB for gaming/media creation 64-128GB for Workstations
Speed
example DDR4-3200mhz|DDR5-6000mhz (check CPU compatibility)
Channels
Dual-channel recommended
Slots
2 or 4 DIMM slots on most motherboards
ECC
Error-Correcting Code for workstations/servers
SO-DIMM
Laptop|compact form factor
Voltage
DDR3L: 1.35 V|DDR3U: 1.25 V|DDR4: 1.2 V|DDR5: 1.1 V

Tuning

Profiles
XMP/EXPO for rated speeds
Latency
CAS timings (lower is better, e.g. 16-18-18-36)
Overclocking
Manual tuning for speed/latency, requires stability testing and voltage regulation
Heatsinks
Improves cooling for high-speed RAM
Undervolting & Overvolting
Can improve stability and lifespan, but may void warranty

Motherboard

Chipset
Determines features (PCIe lanes, Overclock possiblity, USB bus speeds)
Socket
Must match CPU (AMX, LGAXXXX, etc.)
Form Factors
ATX|mATX|ITX

Expansion

PCIe
x16 for GPU, x1/x4 for cards
M.2
Can be NVMe or a Wi-Fi card, external dGPU's is possible using a Thunderbolt adapter which is can be found in laptop mods
Rear I/O
PS-2|USB-C/A|Audio|2.5G LAN|Wi-Fi antennas
BIOS/UEFI
Firmware for hardware config, updates for CPU support and features

Storage

NVMe SSD
Faster startup and better loading times for OS, apps & games; PCIe Gen3/4/5
SATA SSD/HDD
Bulk storage & archives
Best Practice
OS on NVMe + secondary drive for Projects|Documents|Images|Videos|Backups|Games

Backup

Local
External HDD/SSD|ZIP Disks|Tape|NAS
Cloud
Google Drive|OneDrive|iCloud|Dropbox

Cooling

CPU & GPU Coolers
Aircoolers, AIO liquid or custom loop, depends on size, air is preferred but has higher idle temps. water coolers can sustain higher temps over lower time(Model brackets are CPU slot dependent)
GPU Heatsinks
Stock or aftermarket, some high-end GPUs have AIO options, custom loops are recommended
Thermals
Keep CPU/GPU under sustained limits for better performance and duration

Acoustics

Fan Curves
Balance noise vs temps, filters will insulate a little giving less noise but slightly more heat
Dust
Filters & periodic cleaning(For filter use brush, or wash with damp cloth/water)
Case Fans
Intake (front/bottom) & Exhaust (rear/top)
Positive pressure preferred to reduce dust
120mm/140mm common sizes, PWM for speed control
Fan Types
Hydraulic|Magnetic|Sleeve|Ball bearings (longer lifespan)
Static Pressure|Airflow
Case Airflow
Front intake → rear/top exhaust, many configurations possible

Power Supply (PSU)

Wattage
Headroom for GPU spikes and wattage for future upgrades
Efficiency
80+ Bronze/Silver/Gold/Platinum
Cables
Modular preferred; new GPUs may need 12VHPWR or 12V-2x6

Ports & I/O

Front
USB-C/A, Audio combo
Rear
USB-C/A (Gen 2/2x2), HDMI/DP (iGPU dependent), RJ-45
Internal
Headers for USB, RGB, Fans, Audio

Network

Ethernet
10/100/1000 (1GbE), 2.5/5/10GbE
Wi-Fi
802.11ac/ax (Wi-Fi 5/6/6E/7) via M.2 Key E or PCI
Bluetooth
BT 5.x for peripherals

Form Factor

Cases
ATX|mATX|ITX|mITX: size dictates airflow & GPU length
Clearance
CPU cooler height, PSU length, radiator support
Cable Mgmt
Improves airflow & serviceability
